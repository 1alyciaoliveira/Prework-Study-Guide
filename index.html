Prework Study Guide

HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The li tag is used to create bullets on lists.
  • The ul tag is used to list things.
  • The h1, h2 (...) h6 tags are used to order titles.
  • Elements and tags are different, the tags starts and ends parts of an HTML element. The elements enclose the contents in between tags.

CSS

  • CSS is used to style the page.
  • We must create a new file (.css) and link it to the html file.
  • Margin and Padding are different things: margin is on the outside of the box while padding is on the inside.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git checkout branch-name: move to another branch
  • git pull: updates branch
  • We use git add -A command to commit all files in the working brand.
